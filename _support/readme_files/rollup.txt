Converting .mjs (ES Modules) to UMD (Universal Module Definition)
requires a bundler or transpiler because UMD uses a complex function
wrapper that ES Modules cannot generate natively.

1. Using Rollup (Recommended)

Rollup is the industry standard for creating UMD bundles because it
generates clean, minimal code.Install Rollup:

npm install --save-dev rollup

2. Run via CLI:
Replace MyLibrary with the name you want for your global variable in
the browser.

npx rollup input.mjs --file output.umd.js --format umd --name "MyLibrary"
